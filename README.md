# 📚 YerimCepte - Kütüphane Rezervasyon Sistemi

YerimCepte, özellikle sınav dönemlerinde kütüphanelerde yaşanan yer bulma stresini ortadan kaldırmak için tasarlanmış yenilikçi bir mobil rezervasyon ve masa yönetim sistemidir. 

Karadeniz Teknik Üniversitesi (KTÜ) öğrencileri düşünülerek geliştirilmiştir. Sistem, kullanılmayan rezervasyonları otomatik olarak iptal eden algoritmasıyla kütüphane kapasitesinin maksimum verimle kullanılmasını sağlar.

## ✨ Öne Çıkan Özellikler
* **Gerçek Zamanlı Rezervasyon:** Kütüphane planı üzerinden boş masaları anlık olarak görüntüleme ve ayırtma.
* **Oto-İptal Algoritması:** Rezervasyon yapılıp belirlenen süre içinde giriş yapılmayan masaların otomatik olarak boşa çıkarılması.
* **Adil Kullanım:** Aynı kişilerin sürekli yer işgal etmesini önleyen zaman bazlı kısıtlamalar.
* **Hızlı ve Pratik Arayüz:** Öğrencilerin mobil cihazlarından saniyeler içinde işlem yapabileceği kullanıcı dostu tasarım.

## 🛠 Kullanılan Teknolojiler
* **Frontend (Mobil Uygulama):** Flutter
* **Backend & Veritabanı:** Firebase (Cloud Firestore, Firebase Authentication)
* **Gerçek Zamanlı Senkronizasyon:** Firebase Realtime Database / Firestore Snapshot Listeners
